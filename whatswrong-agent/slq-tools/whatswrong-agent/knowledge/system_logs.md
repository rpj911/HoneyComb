# Log Sources and Frequent Problems

## Log Sources
- /usr/g/service/log/syscab_install.log on host: If cannot access to a node or something else wrong with remote nodes, there might be some clues in this log.
- /usr/g/service/log/syscab_diag.log on host: If cannot access to a node or something else wrong with remote nodes, there might be some clues in this log.
- /usr/g/log/ig_obelus.log on igc1: Log for obelus. All Recon behaviors are recorded in this log.
